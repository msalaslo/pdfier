We've made basic examples that show you how to use the PDFier API in all major programming languages. These examples are specifically intended to get you coding in a few minutes without the need to install anything.

Basic Example

PDFier API is very easy to use. It takes a license and either a URL or raw HTML as input and returns a PDF.
You just send a request similar to this, which we will be using in all our examples and which converts http://www.sitetoconvert.com to PDF: 

http://pdfier.com/pdfier-app/urltopdfua?license=yourlicensekey&url=http://www.sitetoconvert.com

You can use a lot of additional parameters in the API call, which give you access to all the features for conversion. These are all specified in the documentation.

NB: You can use both GET as well as POST requests as long as you properly URL encode the data that you pass.

Code example for PHP

You can implement the PDFier API with the file_get_contents function in PHP. Sending the example request and storing the PDF in a local file thus becomes the following code. 

$license = 'yourlicensekey';
$url = urlencode('http://www.sitetoconvert.com');

$result = file_get_contents("http://pdfier.com/pdfier-app/urltopdfua?license=$license&url=$url");
file_put_contents('/tmp/accessiblepdf.pdf',$result);

Of course you can use many options, by referring to the documentation.

Code example for Java

You can use the following code in Java to send the basic example and get the PDF stored as 'accessiblepdf.pdf':

String license = "yourlicensekey";
String url = "http://www.sitetoconvert.com";
File outs = new File("C:\\temp\accessiblepdf.pdf");

URL u = new URL("http://pdfier.com/pdfier-app/urltopdfua?license=" + license + "&url=" + url);
URLConnection uc = u.openConnection();
BufferedInputStream is = new BufferedInputStream(uc.getInputStream());
BufferedOutputStream bout = new BufferedOutputStream(new FileOutputStream(outs));

byte[] b = new byte[8 * 1024];
int read = 0;
while ((read = is.read(b)) > -1) {
	bout.write(b, 0, read);
}
bout.flush();
bout.close();
is.close();

Of course you can use many options, by referring to the documentation.

Code example for Ruby

It is very easy to convert a URL to PDF in Ruby with the PDFier API. Just get the result of our URL with the Ruby class Net::HTTP.

require 'net/http'
license = 'yourlicensekey'
url = 'http://www.sitetoconvert.com'

uri = URI("http://pdfier.com/pdfier-app/urltopdfua?license=#{license}&url=#{url}")
Net::HTTP.start(uri.host, uri.port) do |http|
  request = Net::HTTP::Get.new uri.request_uri

  http.request request do |response|
    open 'accessiblepdf.pdf', 'w' do |io|
      response.read_body do |chunk|
        io.write chunk
      end
    end
  end
end

Of course you can use many options, by referring to the documentation.

Code example for .NET / C#

Converting a URL to PDF in C# is easy with the PDFier API. Just use the C# WebClient method as in the example below. 

string license = "yourlicensekey";
string url = "http://www.sitetoconvert.com";

using (var client = new WebClient())
{
    client.QueryString.Add("license", license);
    client.QueryString.Add("url", url);
    client.DownloadFile("http://pdfier.com/pdfier-app/urltopdfua", @"c:\temp\accessiblepdf.pdf");
}

Of course you can use many options, by referring to the documentation.

Code example for Python

You can use the Python function urllib2.urlopen to make use of our API and convert URLs or HTML to PDF.

import urllib2

data = {
   'license': 'yourlicensekey',
   'url': 'http://www.sitetoconvert.com'
}
requesturl = 'http://pdfier.com/pdfier-app/urltopdfua?license={license}&url={url}'.format(**data)
result = urllib2.urlopen(requesturl)
localFile = open('accessiblepdf.pdf', 'w')
localFile.write(result.read())
localFile.close()

Of course you can use many options, by referring to the documentation.

Code example for Perl

You can use several modules in Perl to use our API, such as the File::Fetch or the LWP::Simple module.

use File::Fetch;

my $license = 'yourlicensekey';
my $url = "http://www.sitetoconvert.com";

my $ff = File::Fetch->new(uri => "http://pdfier.com/pdfier-app/urltopdfua?license=$license&url=$url");
my $where = $ff->fetch( to => '/tmp/accessiblepdf.pdf');

Of course you can use many options, by referring to the documentation.
